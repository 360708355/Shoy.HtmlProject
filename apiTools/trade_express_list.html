获取快递方式列表

根据供货商Id、省份Id、城市Id、商品Id列表获取快递方式列表
1.接口说明
根据供货商Id、省份Id、城市Id、商品Id列表获取快递方式列表
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.express.list
province_id Int 是 省份ID
city_id Int 是 城市ID
factory_id Int 是 供货商ID
product_ids String 是 商品ID列表，多个ID之间以,隔开。
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Json 收货地址信息列表
result[i].post_id Int 快递方式ID
result[i].postway String 快递方式
result[i].fp Float 首重金额
result[i].fw Int 首重单位(g)
result[i].sp Float 续重金额
result[i].sw Int 续重单位(g)
