2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：product.categorys
depth Int 否 深度:0一级分类(默认)、1二级分类、2三级分类。优先级最低
parent_id Int 否 父级分类id(默认0)。优先级其次
cate_id Int 否 指定分类Id(默认0)。优先级最高
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 错误信息
result Json 返回结果，数组对象
result[i].id Int ID
result[i].name String 名称
result[i].depth Int 深度
result[i].parentid Int 父级ID
