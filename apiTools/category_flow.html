获取分类层级

指定分类Id获取出对应的1、2、3及分类
1.接口说明
/// 深度必须介于0-2之间
/// 已指定父级分类id，深度无效
/// 已指定分类Id，父级分类无效
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 Object
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：category.flow
cate_id Int 是 指定分类Id
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 错误信息
result Object 返回结果对象
result.depth Int 当前分类级别：0一级、1二级、2三级。
注：如果当前分类级别为0，则二三级分类没有数据，id默认0、名称默认空字符串。
result.c1_id Int 一级分类ID
result.c1_name String 一级分类名称
result.c2_id Int 二级分类ID
result.c2_name String 二级分类名称
result.c3_id Int 三级分类ID
result.c3_name String 三级分类名称
