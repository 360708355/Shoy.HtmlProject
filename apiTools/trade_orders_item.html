订单详情

根据订单号获取订单详细信息
1.接口说明
根据订单号获取订单详细信息。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.orders_item
ticket String 是 用户令牌，包含登录后的用户信息，登录时获取。
order_num String 是 订单编号，长度为14位
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Object 返回结果
result.order_num String 订单编号
result.payment_type Int 支付方式，1：在线支付；3：余额支付。
result.order_state Int 订单状态，-2：未支付，-1：用户取消，0：未处理；1：正在备货；2：审核失败；3：已经发货；4：发货失败；5：交易完成。
result.products Array 商品列表
result.product[i].id Int 商品ID
result.products[i].num String 商品编号
result.products[i].name String 商品名称
result.products[i].price Double 会员价
result.products[i].price_vip Double vip价，商家登录后，才能获取
result.products[i].count Int 该商品在购物车中的数量，包含这次添加的
result.products[i].wight Int 单个商品重量(g)
result.products[i].pictures Array 商品图片列表：result.pictures[0]为50X50,result.pictures[1]为90X90,result.pictures[2]为120X120
