获取快递方式列表

根据供货商Id、省份Id、城市Id、商品Id列表获取快递方式列表
1.接口说明
根据供货商Id、省份Id、城市Id、商品Id列表获取快递方式列表
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router/verify.shy
格式 JSON
请求方式 GET
3.2 请求参数说明
method 固定为：trade.expresses
province_id 省份ID
city_id 城市ID
factory_id 供货商ID
product_ids 商品ID列表，多个ID之间以,隔开。
3.3 返回参数说明
state 状态：1成功，其他失败
msg 错误信息
result 收货地址信息列表
result[i].postId 快递方式ID
result[i].postWay 快递方式
result[i].fp 首重金额
result[i].fw 首重单位(g)
result[i].sp 续重金额
result[i].sw 续重单位(g)
