/// 获取商品详细信息
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：product.item
product_id Int 是 商品ID
ticket String 否 登录凭证
3.3 返回参数说明
名称 类型 描述
state Int 状态
msg String 错误信息
result Object 返回结果
result.id Int 商品ID
result.num String 货号
result.name String 名称
result.pictures Array 图片路径：
result.pictures[0]为120X120,
result.pictures[1]为160X160,
result.pictures[2]为300X300
result.pic_big Array 多首图：固定300X300，目前仅有1张首图，后续扩展
result.pic_big[0]
result.price_market Double 市场价
result.price Double 会员价
result.price_vip Double VIP价 [仅商家登录有值]
result.brand_name String 品牌名称
result.min_count Int 最低购买数量
result.stock Int 库存
result.iscollect Int 是否已收藏：0否(默认)、1是 [已登录用户有效]
result.isbuy Int 是否可以购买：0否、1是
result.msg String 不能购买原因
result.detail String 商品描述（暂缺）
result.sizes Json 商品尺码集合
result.sizes[i].name String 尺码名称
result.sizes[i].colors Array 当前尺码可选颜色数组
result.sizes[i].colors[i] String 颜色名称
result.colors Array 商品全部颜色数组
result.colors[i] String 颜色名称
