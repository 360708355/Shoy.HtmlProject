生成订单

结算购物车内的商品，生成订单。
1.接口说明
结算购物车内的商品，生成订单。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.orders.generate
ticket String 是 登录凭证，包含登录后的用户信息，登录时获取。
receiver_name String 是 收货人
receiver_mobile String 是 收货人电话
province_id Int 是 省份ID
city_id Int 是 城市ID
receiver_address String 是 街道地址
zip_code String 否 邮政编码
payment_type Int 是 支付方式，1：线上支付；3：余额支付。
coupon_id Int 否 代金券ID
weight Int 是 商品总重量
product_price Double 是 商品总价
postage Double 是 总运费
product_info String 是 商品信息
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Object 返回结果
result.charge_id Int 订单交易号，不同于订单编号，一个交易号可包含多个订单编号。
result.order_nums Array 订单编号，数据对象。
