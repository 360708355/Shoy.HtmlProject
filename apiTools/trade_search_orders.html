获取订单信息
1.接口说明
把商品加入购物车，期间会进行一系列判断，包括库存、活动、商品状态等。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.search_orders
ticket String 是 用户令牌，包含登录后的用户信息，登录时获取。
order_num String 是 订单编号，长度为14位
3.3 返回参数说明
state 状态：1成功，其他失败
msg 错误信息
result 返回结果
result.id 商品ID
result.num 商品编号
result.name 商品名称
result.price 会员价
result.price_vip vip价，user_type为12时，才会有
result.count 该商品在购物车中的数量，包含这次添加的
result.wight 单个商品重量(g)
result.pictures 商品图片列表：result.pictures[0]为50X50,result.pictures[1]为90X90,result.pictures[2]为120X120
