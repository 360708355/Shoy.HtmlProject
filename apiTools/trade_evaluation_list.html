获取商品评价

在交易完成之后，用户可以提交评价。
1.接口说明
在交易完成之后，用户可以提交评价。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为trade.evaluation_list
product_id Int 是 商品Id。
page_no Int 否 默认为1，表示第几页
page_size Int 否 默认为25，表示每页返回多少条数据
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Object 评论列表
result.count Int 评论总数
result.average Double 商品平均得分。
result.list Array 评价列表。
result.list[i].score Int 综合得分。
result.list[i].content String 评价内容。
result.list[i].user_name String 评价人，提供部分信息。
result.list[i].create_on String 评价时间：格式为yyyy-MM-dd。
