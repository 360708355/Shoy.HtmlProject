商品库存

获取商品库存
1.接口说明
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：product.stock
product_id Int 是 商品Id
size String 否 商品尺码(默认 全部尺码)
color String 否 商品颜色(默认 全部颜色)
3.3 返回参数说明
名称 类型 描述
state Int 状态
msg String 错误信息
result Json 返回结果对象
result.stock String 库存数量：充足、少数、缺货、具体库存数字符串
