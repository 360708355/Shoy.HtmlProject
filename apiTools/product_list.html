3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：product.list
cate_id Int 是 分类Id
orderby Int 否 排序方式：0销量降序（默认）、1销量升序、2价格降序、3价格升序
page_no Int 是 当前页码，默认1
page_size Int 是 每页显示数量，默认25
3.3 返回参数说明
名称 类型 描述
state Int 状态
msg String 错误信息
result Object 返回结果
result.count Int 总的条数
result.list Json 结果列表,按销量降序排列
result.list[i].id Int ID
result.list[i].num String 货号
result.list[i].name String 名称
result.list[i].pic_160 String 图片路径
result.list[i].price Double 会员价
result.list[i].price_market Double 市场价
