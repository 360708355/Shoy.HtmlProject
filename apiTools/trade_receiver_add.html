增加常用收货人信息

将收货人信息添加为常用收货人信息。
1.接口说明
将收货人信息添加为常用收货人信息。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.receiver_add
ticket String 是 用户令牌，包含登录后的用户信息，登录时获取。
receiver String 是 收货人
receiver_mobile String 是 收货人电话
province_id Int 是 省份Id
province_name String 是 省份
city_id Int 是 城市Id
city_name String 是 城市
receiver_address String 是 街道地址
post_code String 否 邮编
is_default Bool 否 是否设置为默认收货信息
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
