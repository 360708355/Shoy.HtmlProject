获取收货人信息

根据收货人ID，类型，获取收货人信息
1.接口说明
根据收货人ID，类型，获取收货人信息
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.receivers
ticket String 是 用户令牌，包含登录后的用户信息，登录时获取。
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Json 收货地址信息列表
result[i].Id Int 收货地址信息ID
result[i].UserType Int 用户类型
result[i].UserId Int 用户ID
result[i].Receiver String 收货人
result[i].ReceiverMobile String 收货人电话
result[i].ProvinceId Int 收货人省份ID
result[i].ProvinceName String 收货人省份
result[i].CityId Int 收货人城市ID
result[i].CityName String 收货人城市
result[i].ReceiverAddress String 收货人街道地址
result[i].PostCode String 收货人邮编
result[i].IsDefatul Bool 是否是默认收货地址
