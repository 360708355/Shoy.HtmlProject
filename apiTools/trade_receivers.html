获取收货人信息

根据收货人ID，类型，获取收货人信息
1.接口说明
根据收货人ID，类型，获取收货人信息
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router/verify.shy
格式 JSON
请求方式 GET
3.2 请求参数说明
method 固定为：trade.receivers
user_id 用户ID
user_type 用户类型：1为普通会员，12为商家
3.3 返回参数说明
state 状态：1成功，其他失败
msg 错误信息
result 收货地址信息列表
result[i].Id 收货地址信息ID
result[i].UserType 用户类型
result[i].UserId 用户ID
result[i].Receiver 收货人
result[i].ReceiverMobile 收货人电话
result[i].ProvinceId 收货人省份ID
result[i].ProvinceName 收货人省份
result[i].CityId 收货人城市ID
result[i].CityName 收货人城市
result[i].ReceiverAddress 收货人街道地址
result[i].PostCode 收货人邮编
result[i].IsDefatul 是否是默认收货地址
