用户登录
1.接口说明
/// 支持会员及加盟商登录
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：user.login
name String 是 登录帐号
pwd String 是 登录密码
type Int 否 登录类型：0默认、1会员、2加盟商
mobile_id String 否 手机端唯一标识
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 错误信息
result Object 返回结果
result.utype Int 登录类型：1会员、2加盟商、3会员加盟商均满足，需再次登录，指定请求参数type
result.ticket String 登录凭证
