用户登录
1.接口说明
支持会员及加盟商登录
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
method 固定为：user.login
name 登录帐号
pwd 登录密码
type 登录类型：0默认、1会员、2加盟商
3.3 返回参数说明
state 状态：0失败、1成功
msg 错误信息
result 返回结果
result.utype 登录类型：1会员、2加盟商、3会员加盟商均满足，需深度登录，指定调用参数type
result.session 登录凭证
