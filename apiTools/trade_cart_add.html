加入购物车

把商品加入购物车
1.接口说明
把商品加入购物车，期间会进行一系列判断，包括库存、活动、商品状态等。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.cart.add
ticket String 否 登录凭证，包含登录后的用户信息，登录时获取。未登录用户为空。
product_id Int 是 商品ID
product_size String 是 商品规格
product_color String 是 商品颜色
count Int 是 数量，正数为增加，负数为减少。当购物车中数量小于1，即为删除。
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Object 返回结果
result.id Int 商品ID
result.num String 商品编号
result.name String 商品名称
result.price Double 会员价
result.price_vip Double vip价，商家登录后，才能获取
result.count Int 该商品在购物车中的数量，包含这次添加的
result.wight Int 单个商品重量(g)
result.pictures Array 商品图片列表：
result.pictures[0]为120X120
result.pictures[1]为160X160
result.pictures[2]为300X300
