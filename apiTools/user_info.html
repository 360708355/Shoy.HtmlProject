2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：user.info
ticket String 是 登录凭证
3.3 返回参数说明
名称 类型 描述
state Int 状态
msg String 错误信息
result Object 返回结果
result.utype Int 类型：1会员、2加盟商
result.name String 登录帐号
result.open_id String 第三方登录标识
open_type Int 第三方平台类型：1腾讯QQ、2腾讯微博、3新浪微博、4支付宝。
注：目前不支持所以暂默认0，尽快扩展
result.realname String 真实姓名
result.money Double 余额
result.baifub Double 百福币 [商家]
result.coupon Double 代金券 [会员]
result.grade Int 商城等级 [商家]
result.mobile String 手机号
result.qq String QQ号
