/// 支持会员及加盟商注销登录
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：user.loginout
ticket String 是 登录凭证
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 反馈信息：会员已注销、加盟商已注销
3.4 附
注销时可直接清除终端ticket、或者调用此接口通过服务器清除
