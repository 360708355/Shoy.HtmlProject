3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：user.register
name String 是 帐号
pwd String 是 密码
realname String 否 真实姓名，可为空字符串
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 错误信息
result Json 返回结果
result.ticket String 登录凭证。
