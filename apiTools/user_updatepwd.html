修改密码
1.接口说明
/// 支持会员、加盟商修改密码及密码找回
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：user.updatepwd
name String 否 帐号 [找回密码使用]
type Int 否 帐号类型：1会员、2加盟商 [找回密码使用]
code String 否 验证码 [找回密码使用]
ticket String 否 登录凭证 [修改密码使用]
newpwd String 是 新密码
3.2 附
找回密码使用相关参数及修改密码使用参数不能同时为空
3.3 返回参数说明
名称 类型 描述
state Int 状态：0失败、1成功
msg String 提示信息
