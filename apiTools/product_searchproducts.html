搜索商品
1.接口说明
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
method 固定为：product.searchproducts
keyword 搜索关键词
cateId 三级分类Id，默认0
page_no 当前页码，默认1
page_size 每页显示数量，默认25
3.3 返回参数说明
state 状态
msg 错误信息
result 返回结果
result.count 总的条数
result.list 搜索列表
result.list[i].id ID
result.list[i].num 货号
result.list[i].name 名称
result.list[i].pic160 图片路径
result.list[i].price 会员价
result.list[i].pricemarket 市场价
