App版本检测

App版本检测接口，用于App提供升级检测。。
1.接口说明
App版本检测接口，用于App提供升级检测。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 GET
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：other.version_detected
app_type Int 否 App类型，默认为0,Android App;后续其他
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
result Object 返回结果信息
result.version String 版本号
result.upgrade_instructions String 升级说明
result.download_url String 下载地址
result.mandatory Bool 是否强制升级
result.upgrade_date String 版本发布日期,格式为：yyyy-MM-dd
