更新订单状态

更新订单状态，目前主要用于取消订单、确认收货。
1.接口说明
更新订单状态，目前主要用于取消订单、确认收货。
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router.verify
格式 JSON
请求方式 POST
3.2 请求参数说明
名称 类型 是否必须 描述
method String 是 固定为：trade.orders.update
ticket String 是 登录凭证，包含登录后的用户信息，登录时获取。
order_num String 是 订单编号，长度为14位
status_code Int 是 新订单状态：-1为取消订单；5为确认收货。
3.3 返回参数说明
名称 类型 描述
state Int 状态：1成功，其他失败
msg String 错误信息
