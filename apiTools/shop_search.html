搜索店铺
1.接口说明
根据关键字搜索店铺
2.应用场景
3.接口调用说明
3.1 请求说明
url http://www.api.100hg.com/apiTools/router/verify.shy
格式 JSON
请求方式 GET
3.2 请求参数说明
method 固定为：shop.search
key 搜索关键字
page_no 默认为1，表示第几页
page_size 默认为25，表示每页返回多少条数据
3.3 返回参数说明
state 状态：1成功，其他失败
msg 错误信息
result 返回结果
result.count 总的条数
result.list 结果列表
result.list[i].id 店铺ID
result.list[i].sub 店铺二级域名
result.list[i].name 店铺名称
